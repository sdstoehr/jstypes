Truthy and falsy values in JavaScript

In JavaScript there are the two boolean values "true" and "false". But you can use any other data type as a condition. These data types have an implicitly assigned boolean value and are considered either "truthy" or "falsy".

Falsy values

The following values are considered "falsy":

  • false
  • 0
  • ""
  • null
  • undefined
  • NaN

Truthy values

All other values are considered "truthy". For example:

  • true
  • non-zero numbers
  • non-empty strings (e.g. "abc", "0", "false", ...)
  • functions
  • (empty) arrays
  • (empty) objects
